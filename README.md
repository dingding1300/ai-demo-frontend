# AI图像生成演示项目

## 项目说明
本项目是一个AI图像生成前端演示项目，满足以下作业要求：
1.  前端界面显示学号和姓名
2.  支持参数调整（提示词、图片尺寸、生成步数）
3.  可迁移、可优化、可扩展
4.  属于AI前沿技术项目

## 项目功能
- 输入提示词生成AI图像
- 调整图片尺寸、生成步数等参数
- 展示生成结果

## 运行方式
1.  克隆仓库：`git clone https://github.com/dingding1300/ai-demo-frontend.git`
2.  直接用浏览器打开 `index.html` 即可运行
3.  （可选）替换 `script.js` 中的模拟API为真实AI图像生成API
